## Interactive Elements

### Buttons:
- Primary (Download/Confirm): Blue background, white text, hover darker
- Secondary (Cancel): Light gray background, dark text
- Disabled: Opacity 50%, cursor not-allowed

### Checkboxes:
- Unchecked: White background, gray border
- Checked: Blue background, white checkmark
- Hover: Slight border color change

### Dropdowns:
- Default: White/dark background, border
- Open: Dropdown menu below with options
- Selected: Blue highlight on option
- Hover: Light background on option

### File Upload Area:
- Default: Dashed border, light background
- Hover: Blue border, slightly lighter background
- Dragging: Blue border, blue tinted background
- Loading: Spinner centered

## Responsive Behavior

### Desktop (lg+):
- Full width layout
- Field selector in 4 columns
- Filters in 3 columns side-by-side
- Modals: Max width 5xl (80rem)

### Tablet (md):
- Field selector in 3 columns
- Filters in 2 columns (Status wraps)
- Modals: Max width 4xl (56rem)

### Mobile:
- Field selector in 2 columns
- Filters stack vertically (1 column)
- Modals: Full width with margins
- Tables scroll horizontally

## Accessibility Features

- Keyboard navigation supported
- Focus indicators on all interactive elements
- Screen reader friendly labels
- Error messages announced
- Color contrast meets WCAG AA standards
- Loading states clearly indicated
