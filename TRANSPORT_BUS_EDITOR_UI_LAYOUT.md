## Search Functionality

The search bar filters buses in real-time based on:
- Bus number (e.g., "001")
- License plate (e.g., "ABC-1234")
- Driver name (e.g., "John")
- Campus name (e.g., "Main")

Search is case-insensitive and shows results as you type.
